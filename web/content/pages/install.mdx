# Install Territory Developer

Territory Developer is in active development. Pre-release builds are not yet publicly distributed. Platform availability and access channels are listed below.

## System Requirements

- **OS:** Windows 10+, macOS 12+, or Ubuntu 22.04+
- **RAM:** 4 GB minimum, 8 GB recommended
- **Storage:** 500 MB
- **GPU:** Any GPU supporting OpenGL 3.3+ or Metal

## How to Get Access

Early access will be announced via the project's public channels. Until then, you can follow progress on the [history page](/history) or check back here for release dates as each platform reaches a distributable state.
